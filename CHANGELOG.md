## [1.3.0] — 2026-06-26 (5A 升级 · M4-C 元进程 / 解锁)

### 新增 — 元进程 / 解锁系统
- **累计统计存档**（v2→v3 自动迁移）：累计击杀、通关次数、游玩局数，标题页与图鉴展示。
- **解锁系统**（由统计推导，自洽不丢档）：
  - **角色皮肤** 4 款：士兵·绿(初始) / 游侠·橙(通关1次) / 暗影·紫(击杀200) / 黄金·金(最高分8000)。
  - **初始武器** 5 款：弓箭(初始) / 机关枪(通关1次) / 喷射枪(击杀120) / 烈焰弹(击杀320) / 激光(通关2次)——开局即可携带，不必等关内拾取。
- **解锁图鉴**：标题页「解锁图鉴」查看全部皮肤/武器的解锁状态与条件。
- **结算页新解锁提示**：本局达成的解锁实时显示「🔓 新解锁：…」。

### 改进 — 标题流程
- 标题页重做为**光标菜单**：开始 / 难度 / 人数 / 初始武器 / 角色皮肤 / 解锁图鉴，↑↓ 选择、←→ 调整、Enter/点击确认；保留 Shift 快捷切人数。
- 失败重试视为新一局：重置本局统计并重新应用所选初始装备。

## [1.2.0] — 2026-06-26 (5A 升级 · M4-B 本地双人合作)

### 新增 — 本地双人合作 (2P Co-op)
- 标题页 **Shift 一键切换单人/双人**；双人同屏，P1（绿）/P2（蓝）不同配色 + 头顶 P1/P2 标记。
- **双人键盘分区**：P1 = `WASD` 移动/瞄准 · `空格` 跳 · `左Shift`/`F` 射击；P2 = `↑↓←→` · `/` 跳 · `.`/`右Shift` 射击。
- **双手柄**：GP0→P1、GP1→P2，各自独立。
- **共享命数池** + **濒死救援**（队友存活时在其身边复活）；全部出局才 Game Over。
- **跟随双人中点的智能镜头**；敌人/Boss/炮台**就近索敌**最近的存活玩家。
- HUD 升级：左右双血条 + 各自武器图标 + 居中共享余命。

## [1.1.0] — 2026-06-26 (5A 升级 · M4-A 手感/演出)

向 5A 标准推进的第一轮：手感、演出、难度、无障碍、手柄。

### 新增 — 打击感 (Game Feel)
- **trauma 平方曲线相机震动** + **开火相机踢**（按瞄准方向反冲），比线性抖动更有冲击力。
- 武器**后坐力** + **枪口闪光粒子** + **弹道拖尾(tracer)** + **弹壳抛射** + **命中迸溅火花**；各武器手感差异拉开。
- 命中/击杀/爆炸的**顿帧(hit-stop)**调优，击杀有微震 + 弹反白闪。
- 敌人命中**击退**反馈；拾取/击败更强的闪光与飘字。

### 新增 — 难度与无障碍
- **三档难度**（简单/普通/困难）：影响敌人血量/弹速/射频/玩家受伤与命数；标题页与暂停菜单均可切换。
- **无障碍**：色盲友好（敌弹高对比描边+品红）、减弱动效（关闭震动/闪光/暗角）。
- 存档**版本化**（v1→v2 自动迁移）。

### 新增 — 平台与视听
- **手柄支持**（Gamepad API）：左摇杆/方向键移动瞄准、A 跳、X/RT 射击、Start 暂停。
- **立体声像**：音效按世界位置左右声相；更丰富的程序化音效。
- **暗角(vignette)** 氛围层 + **关卡淡入转场**。

### 下一轮 (规划中)
- M4-B 本地双人合作 + M4-C 元进程/解锁 + M4-D 视听深化与精灵动画。

## [1.0.0] — 2026-06-25

首个可交付版本。

### 核心玩法
- 横版**跑射**：跑动、二段跳、**8 方向瞄准**（仰射/斜射/向下射/卧倒平射）。
- **武器系统**：默认弓箭 + 喷射(S)/机枪(M)/激光(L,穿透)/烈焰(F)，道具拾取切换；死亡重置为默认。
- **敌人**：僵尸、骷髅、苦力怕（自爆）、炮台（三连扫射）、烈焰人（飞行火球）。
- **3 关 + 3 Boss**：丛林前线→下界要塞核心；下界基地→凋灵；末地核心→末影龙。各含血条+数字、低血强化、击败掉传送门过关。
- **命数系统**：3 命 × 4 心，安全点复活，命尽 Game Over；连击计分、最高分存档。

### 工程（工业级底座）
- 固定时间步长主循环（帧率无关）。
- 高 DPI 超采样渲染（4K/5K 无毛刺）。
- 移动端全局触摸追踪（多指/滑动/横屏可用，`touch-action:none`）+ 十字方向键 + 瞄准 + 跳/射/暂停按钮。
- 暂停/设置菜单（音量/静音/画面震动）、localStorage 存档、失焦自动暂停、崩溃保护层、上下文丢失恢复。
- 程序化 SFX + 每关背景音乐（Boss 战切激烈曲）、命中顿帧/全屏闪光/伤害飘字打击感。
- 交付工具链：`npm start`（服务器）、`npm run build`（单文件）、`npm test`（headless 冒烟测试 + 截图，12 项断言全过、0 错误）。
- `?debug` QA 钩子（`__c.warpToBoss()` / `__c.give()` / `__c.nuke()`）。
