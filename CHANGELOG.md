# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] — 2026-06-25

首个可交付版本。

### 核心玩法
- 横版**跑射**：跑动、二段跳、**8 方向瞄准**（仰射/斜射/向下射/卧倒平射）。
- **武器系统**：默认弓箭 + 喷射(S)/机枪(M)/激光(L,穿透)/烈焰(F)，道具拾取切换；死亡重置为默认。
- **敌人**：僵尸、骷髅、苦力怕（自爆）、炮台（三连扫射）、烈焰人（飞行火球）。
- **3 关 + 3 Boss**：丛林前线→下界要塞核心；下界基地→凋灵；末地核心→末影龙。各含血条+数字、低血强化、击败掉传送门过关。
- **命数系统**：3 命 × 4 心，安全点复活，命尽 Game Over；连击计分、最高分存档。

### 工程（工业级底座）
- 固定时间步长主循环（帧率无关）。
- 高 DPI 超采样渲染（4K/5K 无毛刺）。
- 移动端全局触摸追踪（多指/滑动/横屏可用，`touch-action:none`）+ 十字方向键 + 瞄准 + 跳/射/暂停按钮。
- 暂停/设置菜单（音量/静音/画面震动）、localStorage 存档、失焦自动暂停、崩溃保护层、上下文丢失恢复。
- 程序化 SFX + 每关背景音乐（Boss 战切激烈曲）、命中顿帧/全屏闪光/伤害飘字打击感。
- 交付工具链：`npm start`（服务器）、`npm run build`（单文件）、`npm test`（headless 冒烟测试 + 截图，12 项断言全过、0 错误）。
- `?debug` QA 钩子（`__c.warpToBoss()` / `__c.give()` / `__c.nuke()`）。
