# 🔫 MC 魂斗罗 — 我的世界版魂斗罗

一个用 HTML5 Canvas 写的横版**跑射闯关**游戏（纯前端，零依赖）。魂斗罗式的 8 方向瞄准 + 多武器，换上《我的世界》皮肤：主角是 Steve 大兵，敌人是 MC 怪物与炮台，关底是 MC Boss。

## ▶ 怎么玩

**最简单**：双击 `MC魂斗罗-单文件版.html`，浏览器打开即玩，零安装、零依赖、可离线。

> 第一次有声音前需先按键 / 点屏（浏览器策略）。

也可双击 `index.html`（外壳 + `game.js`），或用本地服务器（见「开发」）。

## 🎮 操作

| 操作 | 按键 | 手机 |
|------|------|------|
| 移动 | `←` `→` / `A` `D` | 左侧 ◀ ▶ |
| 跳跃（可二段） | `空格` | ⤒ |
| 瞄准上 / 斜上 | `↑`（配合左右） | ▲ |
| 瞄准下 / 卧倒 | `↓`（地面=卧倒，空中=向下射） | ▼ |
| 射击（**按住连发**） | `J` / 鼠标左键 | 🔫 |
| 暂停 / 设置 | `P` / `Esc` | ⏸ |
| 快速静音 | `M` | — |
| 开始 / 重试 | `Enter` / 点屏 | 点屏 |

**8 方向瞄准**：站立时按 `↑` 仰射、`↑`+方向斜射；空中按 `↓` 向下射、`↓`+方向斜下射；地面按 `↓` 卧倒（低姿态平射）。

## 💾 存档 & 体验

- 命数 3 条，每条 4 颗心；死亡在镜头左侧安全地面复活，命尽则 Game Over。
- 历史最高分、设置（音量/静音/震动）存浏览器本地，自动恢复。
- **打击感**：命中顿帧、全屏闪光、伤害飘字、连击计分。
- **每关程序化背景音乐**，Boss 战自动切更激烈的曲子，受全局音量/静音控制。
- 失焦自动暂停；摄像机带朝向前瞻 + 平滑跟随。

## 🛠 技术

- **帧率无关**：逻辑固定 60Hz 步进，渲染随刷新率（120/144Hz 速度一致）。
- **高 DPI 无毛刺**：画布按设备物理像素超采样渲染，4K/5K 文字图形清晰。
- **移动端健壮触控**：全局触摸坐标追踪，多指/滑动/横屏均可用（`touch-action:none`）。
- 关卡是 `game.js` 顶部 `LEVELS` 里的字符画地图，改图直接编辑字符串（图例见注释）。

## 🧑‍💻 开发 / 构建 / 测试

需要 Node.js 18+：

```bash
npm start              # 本地服务器 http://localhost:8081
npm run build          # 生成单文件版 (dist/ + 根目录)
npm test               # headless 冒烟测试 + 截图 (需先 npx playwright install chromium)
npm run check          # game.js 语法检查
```

> 改完 `game.js` / `index.html` 后 `npm run build` 重新生成单文件版。

## 📦 交付

- 发给朋友：单独发 `MC魂斗罗-单文件版.html` 一个文件即可。
- 静态托管：上传 `dist/`，入口 `index.html`。
